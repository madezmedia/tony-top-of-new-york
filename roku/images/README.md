# Roku App Images

This directory contains all branding images for the TONY Roku app.

## Required Images

### App Icons

#### HD Icon (290x290px)
- **Filename:** `icon-hd.png`
- **Size:** 290x290 pixels
- **Format:** PNG with transparency
- **Usage:** Home screen, channel launcher
- **Design:** TONY logo or character face, bold, recognizable at small size

#### SD Icon (214x144px)
- **Filename:** `icon-sd.png`
- **Size:** 214x144 pixels
- **Format:** PNG with transparency
- **Usage:** Older Roku devices
- **Design:** Same as HD icon, just cropped/letterboxed

#### Focus Icon (290x290px)
- **Filename:** `icon-focus-hd.png`
- **Size:** 290x290 pixels
- **Format:** PNG with transparency
- **Usage:** Highlighted state in home screen
- **Design:** Same as HD icon but with glow/border for focus state

### Splash Screen

#### HD Splash (1920x1080px)
- **Filename:** `splash-hd.jpg`
- **Size:** 1920x1080 pixels
- **Format:** JPG (no transparency)
- **Display time:** 1500ms (1.5 seconds)
- **Background color:** #020B13 (dark blue/black)
- **Design:** TONY logo, "Top of New York" tagline, dramatic NYC skyline or character art

### Optional Images

#### Overhang Banner (1920x300px)
- **Filename:** `overhang.png`
- **Size:** 1920x300 pixels
- **Format:** PNG with transparency
- **Usage:** Top banner that appears above all screens
- **Design:** TONY logo on left, "Top of New York" text on right

#### Poster/Backdrop (1920x1080px)
- **Filename:** `poster-fhd.jpg`
- **Size:** 1920x1080 pixels
- **Format:** JPG
- **Usage:** Channel store listing, detail view
- **Design:** Dramatic scene with TONY character, NYC backdrop

## Design Guidelines

### Colors
- **Primary:** #020B13 (dark blue/black background)
- **Accent:** Gold/yellow for energy, premium feel
- **Text:** White for readability

### Typography
- **Font:** Bold, sans-serif
- **Style:** Urban, edgy, NYC vibes
- **Tagline:** "Top of New York" - prominent

### TONY Character
- Face-forward portrait or slight angle
- Expression: Confident, street-smart, determined
- Style: Streetwear, hoodie, NYC aesthetic

### NYC Elements
- Skyline silhouette
- Subway map references
- Street art/graffiti touches
- Borough references (Harlem, Brooklyn, etc.)

## Tools to Create Images

1. **Canva** - Free templates for Roku icons
2. **Figma** - Professional design tool
3. **Photoshop/Illustrator** - If you have Adobe
4. **Remove.bg** - Remove backgrounds from photos
5. **TinyPNG** - Optimize PNG file sizes

## Quick Start

1. Create or source a TONY logo/character image
2. Resize to required dimensions (use Canva, Figma, or ImageMagick)
3. Save in this directory with correct filenames
4. Rebuild the zip and sideload to test
